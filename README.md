# Wake Transmission

Wake Transmission is an audio-reactive 3D visualizer built with React, Three.js, and Web Audio. It turns music into animated landscapes, HUD overlays, and theme-driven motion.

## Highlights

- Frequency-reactive 3D geometry
- Multiple visual themes and animation modes
- Built-in soundscape playback and immersive overlays
- Motion-heavy interface built for fullscreen listening sessions

## Stack

- React
- Three.js
- Framer Motion
- Tone.js

## Quick start

```bash
bun install
bun run dev
```

Checks and build:

```bash
bun run typecheck
bun run build
```
